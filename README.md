# Customer Database Project

## What This Project Does
This project is a simple customer database management system. It stores customer information imported from a CSV/Excel file into a MySQL database. The project allows viewing, editing, and managing customer records using phpMyAdmin and MySQL.

## How I Used XAMPP and MySQL
I used XAMPP to create a local server environment on Windows. Apache and MySQL services were started from the XAMPP Control Panel. Using phpMyAdmin, I created a database and imported customer data from a CSV file into a MySQL table. I also learned how to modify table columns, edit records, and manage the database structure.

## How I Used ChatGPT as a Coding Assistant
I used ChatGPT to help me learn database concepts, MySQL syntax, and XAMPP setup. ChatGPT assisted me with importing CSV files, creating database tables, editing data in phpMyAdmin, and understanding common errors while learning web development and database management.
